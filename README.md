# Git 使用总结

## 基本概念

- 工作区
- 暂存区
  +临时备份
  +分批次提交
- 版本库（每一次的 commit 都提交到了版本库）

## 基本操作

-git init [仓库名称]
-git add  工作区提交到暂存区
-git commit -m 提交日志
-git commit -a -m 提交工作区自上次 commit 之后的变化，直接到仓库区
-git status 查看当前状态
-git log 
-git rm --cached 文件名
 +移除被 git 跟踪管理的文件
